## Журнал решений (кратко, для истории)

### Session 2 (2026-06-05) — построили фундамент Website Department
- **Метод (Марина):** сначала структура из конвертящих конкурентов (эмпирика) → потом обучение (Елена и
  т.д.) как линза для усиления. Не «всё в кучу».
- **Структура карточки (9 обяз. + До-После опц.):** Галерея · Buy-box · UGC (наверх) · О товаре · До-После ·
  Мы-Они · Инструкция · FAQ · Оффер-ещё-раз · Written-отзывы (в конец). «О бренде» — нативно + в Базе.
- **Stage 0 → REV1:** первый прототип из знаний Claude.
- **Stage 0 → REV2:** перестановка ритма (UGC наверх, отзывы разделены: UGC ↑ / written ↓, хвост
  FAQ→оффер→отзывы) — убрали «кучу картинок подряд».
- **Stage 0 → REV3:** перекраска под Be Yours (изучили чистую тему: button-radius 0, флэт-карточки,
  Cabin/Jost, секции встык). Теперь читается как Shopify-сборка.
- **Философия темы (Марина):** Be Yours = инструмент доставки (checkout/аналитика/продукт завязаны на
  платформу). Дизайн — наш; где сделаем круче темы — делаем сами и вставляем кусками; функциональные блоки
  (корзина и т.п.) берём нативные.

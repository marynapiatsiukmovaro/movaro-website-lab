Stage 2 · структура

Несвязанная экспертная версия

Эмоциональный hero text-on-photo, Fraunces+Jost, новый блок «Почему это успокаивает» (механизм), 3-кол. сравнение, голос ветеринара, рабочий выбор бандла/размера → цена, sticky, scroll-reveal.
2.1 · Calm Clinical

Спокойный клинический — петроль

Костяной фон + графит + пыльно-петролевый акцент. Движение — «тихая роскошь» (hero + «дыхание» кота). Fraunces.
2.2 · Warm Editorial

Тёплый журнальный — терракота

Слоновая кость + эспрессо + терракота. Движение — бегущая строка + редакторские reveal. Playfair Display.
2.3 · Mono + Honey

Смелый моно + мёд — кинематографичный

Off-white + тёмные секции + мёд. Движение — параллакс, hotspots на фото врапа, drag-слайдер до/после. Space Grotesk.
2.4 · + вовлечение

Calm Clinical + слой вовлечения

База 2.1 + hotspots на врапе + «пульс-линия» через страницу (паника → спокойствие, едет шарик) + точечный хайлайт/эмодзи. До/После — side-by-side. Исследование, не выбор.
